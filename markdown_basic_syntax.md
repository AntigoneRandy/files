表格的语法是
```markdown
|表头|表头|表头|
|---|:--:|---:|
|内容|内容|内容|
|内容|内容|内容|

第二行分割表头和内容。
- 有一个就行，为了对齐，多加了几个
  :- 文字默认居左
 :-: 表示文字居中
  -: 表示文字居右
```

实例:

|姓名|技能|实力|
|---|:--:|---:|
|vv|啥都会|dalao|
|hyy|啥都会|dalao|
|lbh|啥都不会|five|

</br>

|默认是左对齐|我也是左对齐|我是中对齐|我是右对齐|
|-----------|:----------|:-------:|--------:|
|左左左左|俺也一样|啦啦啦|往右对齐看起来|
|左左左左|俺也一样|中对齐|好不爽啊|
|左左左左|俺也一样|就是我|俺也觉得|
</br>
